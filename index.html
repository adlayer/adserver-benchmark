    Este repositório contem a estrutura necessária, para a realização de uma comparação de performance entre plataformas de adserving.

    Estrutura

    Cada plataforma testada possui uma pasta exclusiva, com os arquivos necessários para renderizar uma página web simples. Todas as integrações utilizam como base a mesma estrutura HTML e o código dos espaços são inseridos na exata mesma posição.

    Testes

    A estrutura fornecida neste repositório, sugere que se leve em conta o tempo de carregamento de todos os recursos, afim de mensurar o a velocidade que as peças aparecem e seu impacto na performance da página.

    Em breve novas comparações serão adicionadas.
